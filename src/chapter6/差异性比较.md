# 差异性比较（git diff）
diff命令也是Linux操作系统自带的一个命令，就像rm、mv等命令一样，git将这些命令也纳入到了自己的命令中

```bash
# 显示工作区与暂存区之间的差异
git diff

# 显示最新的提交与工作区与的差异
git diff HEAD 

# 显示最新的提交与暂存区的差异
git diff --cached [commit]

# 显示两次提交之间的差异
git diff [first-branch]...[second-branch]

# 显示今天你写了多少行代码
git diff --shortstat '@{0 day ago}'
```
