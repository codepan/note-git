## 远程仓库 (git remote)
```shell
# 查看远程仓库
git remote

# 查看远程仓库，并显示更加详细的信息
git remote -v

# 显示所有或者某个远程仓库的信息
git remote show [remote]

# 增加一个新的远程仓库，并命名
# eg: git remote add origin <url> 本地仓库与远程仓库作关联，其中origin是远程仓库的默认名字。含义是：给远程仓库起一个别名，名字叫做origin
git remote add <shortname> <url>

# 删除xxx远程仓库别名
git remote remove <仓库名称>
```
